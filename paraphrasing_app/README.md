# AI Paraphrasing Web Application

## 🚀 Overview
This project is an AI-powered paraphrasing web application that generates multiple variations of a sentence using NLP and Machine Learning techniques.

## 🧠 Features
- Professional, Expressive, and Casual paraphrasing styles
- Grammar correction and input cleaning
- Diversity filtering and ranking system
- CSV-based dataset collection for future training
- Rating system (Good/Bad outputs)
- Flask-based REST API
- Interactive frontend (HTML, CSS, JavaScript)

## 🛠️ Tech Stack
- Python
- Flask
- HuggingFace Transformers
- PyTorch
- HTML, CSS, JavaScript

## 📊 ML Pipeline
1. User inputs sentence
2. Model generates paraphrases
3. Outputs filtered and ranked
4. Stored in CSV dataset
5. Dataset used for future fine-tuning
